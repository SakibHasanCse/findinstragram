Find Instagram User ID

How to use?
Enter your Instagram username in the text box. For example: ladygaga, 9gag, etc. Press enter or click the blue "Find Instagram User ID" button. Your Instagram user ID, profile image and name will appear inside the green box. What is this tool? This tool called "Find Instagram User ID" provides an easy way for developers and designers to get Instagram account numeric ID by username.
